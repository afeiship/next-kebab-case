# next-kebab-case
> Kebab case for next.

## installation
```bash
npm install -S @feizheng/next-kebab-case
```

## usage
```js
import '@feizheng/next-kebab-case';

const res = nx.kebabCase('raDiuScHan_ge d');

// ra-diu-sc-han-ge-d
```
