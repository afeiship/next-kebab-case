# next-kebab-case
> Kebab case for next

## install:
```bash
npm install -S afeiship/next-kebab-case --registry=https://registry.npm.taobao.org
```

## usage:

```js
import 'next-kebab-case';
nx.kebabCase('raDiuScHan_ge d');
// ra-diu-sc-han-ge-d
```
